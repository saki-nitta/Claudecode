# TOY通貨システム設計書 v1

作成日: 2026-05-08  
状態: draft  
前提: 1TOY = 1円・会場内限定・イベント完結型

---

## 基本設計

```
【TOYの入手方法】
  A. おもちゃ持ち込み → 査定 → TOY付与
  B. 現金 → 1円 = 1TOY で購入

【TOYの使い方】
  会場内のおもちゃと交換（新田さん仕入れ品 + 参加者持ち込み品）

【イベント終了後】
  未使用TOYは失効 → 実質参加費として徴収
```

---

## 査定基準・TOY付与額（参加者持ち込み）

### チェック項目（クロ提供・2026/5/8）
査定時は以下の4点を順番に確認する：
1. **清潔さ**：汚れ・ホコリの有無
2. **付属品の有無**：箱・説明書・パーツが揃っているか
3. **動作確認**：電池ものは実際に動かす
4. **安全性**：PSCマークの有無（対象品の場合）

### TOY付与レンジ（クロ更新案・2026/5/8確定）

参加者持込は**状態のみ**で査定。ブランド不問。シンプルに説明できる基準。

| ティア | 状態の目安 | TOY付与レンジ |
|---|---|---|
| **S** | 新品同様・完品 | 300〜500 TOY |
| **A** | 美品・動作OK | 200〜300 TOY |
| **B** | 使用感あり | 100〜200 TOY |
| **C** | 傷・欠品あり | 50〜100 TOY |
| **NG** | 破損・衛生上の問題 | 受付不可 |

※ 参加者持込のTOY上限は500。仕入れ品との価格バランスを保つため。  
※ 仕入れ品の値付けは別途「状態×ブランド」基準で設定（下記参照）。

## 新田さん仕入れ品の値付け基準（TOY建て）

仕入れ品はイベント前に新田さんが**メルカリ相場の60〜70%**をTOY価格として設定する。

```
TOY価格の計算式：
  メルカリ中古相場（同状態の売れた価格） × 0.65 ≒ 販売TOY価格
  ※ 端数は50TOY単位に丸める
```

|---|---|---|---|---|
| T001 | KAPLA 188ピース 欠品あり | ¥3,500〜5,000 | **2,800 TOY** | 1,400〜1,960 TOY |

## 買戻しレート（参加者が以前購入した品を持ち込む場合）

新田さん仕入れ品を買って、次回バトン市に持ち込んだ場合：

```
買戻しTOY = 販売TOY価格 × 50〜70%
  （状態が良いほど70%寄り、劣化があれば50%寄りで査定）
```

## 収益シミュレーション（試算例）

### 前提
- 参加者: 20名
- 持ち込み: 1人平均3点（Bティア想定・150TOY/点）
- 現金TOY購入: 1人平均300TOY分（¥300）
- 新田さん仕入れ品の売上: 10点 × 平均1,000TOY

### 試算

| 項目 | TOY | 円換算 |
|---|---|---|
| 参加者持ち込みで発生したTOY | 20名×3点×150 = 9,000 TOY | — |
| 現金TOY購入 | 20名×300 = 6,000 TOY | **¥6,000** |
| 新田さん仕入れ品の販売 | 10点×1,000 = 10,000 TOY受取 | — |
| 未使用TOY（想定10%） | 1,500 TOY失効 | **¥1,500**（参加費分） |
| **現金収入合計** | — | **¥7,500** |
| 仕入れコスト（10点分） | — | 約¥5,000 |
| **粗利（試算）** | — | **約¥2,500** |

※ これは最小規模の試算。参加者数・仕入れ品の質・現金TOY購入率で大きく変わる。
