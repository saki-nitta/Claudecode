## S④ これを選ぶ理由

**見出し（セクション）**：`これを選ぶ理由`

**理由①**
```
01　世界にひとつ、その子だけのもの

同じ名前でも、同じ体重でも、生まれた瞬間が違えば模様は変わります。
量産品では作れない、受注生産だからできる贈りもの。
```

**理由②**
```
02　「かわいい」で終わらない記念品

写真は劣化する。洋服はサイズアウトする。
木は経年変化しながら、何十年も手元に残ります。
出産祝いに、誕生記念に。飾っても、遊んでも、残しても。
```

**理由③**
```
03　岩手の職人の手仕事

素材の調達から加工・研磨・梱包まで、
すべて岩手の人の手が関わっています。
贈るたびに、地域の森と職人を支えることにもなります。
```

**理由④**
```
04　安心して渡せる品質

食品グレードの塗装、こどもPSCマーク申請予定。
赤ちゃんに渡す前提で、素材から仕上げまで設計しています。
```

## S⑥ 模様が生まれる仕組み

**見出し（セクション）**：`模様が生まれる仕組み`

**STEP 1**
```
データを入力する

Webアプリに赤ちゃんの7つの情報を入力します。
```

**STEP 2**
```
自然計算で模様を生成する

アルゴリズムが数値を自然のパターンに変換。
同じ模様は、二度と生まれません。
```

**STEP 3**
```
岩手の木に刻む

生成された模様をNC加工またはレーザー刻印で、
9枚の葉形パネルそれぞれに刻みます。
```

**STEP 4**
```
手元に届く

9樹種の葉形パネルが揃ったセットとして、
丁寧に梱包してお届けします。
```

## S⑦ 岩手の木である理由

**見出し**：`この子が生まれた土地の、木を使う。`

**本文**：
```
てもりの素材は、岩手県産材9樹種。
スギ、カラマツ、アカマツ、ナラ、ヤマザクラ、
クリ、オニグルミ、イタヤカエデ、ケヤキ。

盛岡の森から切り出され、フローリング材として
丁寧に加工された15mm厚の板を使っています。
それぞれの木が持つ色・木目・硬さ・香りが違う。
9枚並べると、岩手の森が手の上に広がります。

地域の木を、地域の職人が加工し、
赤ちゃんのもとへ届ける。
てもりは、岩手の森とこの子をつなぐものでもあります。
```

**樹種タグ（9個）**：スギ　カラマツ　アカマツ　ナラ　ヤマザクラ　クリ　オニグルミ　イタヤカエデ　ケヤキ
