## 1. Who & Why（このプロジェクトについて）

- **クライアント**：とうめいしっぽ（おもちゃと遊び企画舎・代表：新田紗希／公認心理師・臨床心理士）
- **ミッション**：遊びで親子をつなぎ、世界を平和にする。
- **タグライン**：あそびが、こころにふれる。
- **制作目的**：2026年秋のMakuakeクラウドファンディングに向けた、ブランド認知拡大とコア商品「てもり」の魅力を伝えるWebサイト制作。
- **ターゲット**：20〜30代の子育て世代（ママ・パパ）。スマホからの閲覧が9割を占める想定。

---

## 2. Where & How（制作環境と進め方の方針）

- **使用ツール**：ノーコードWeb制作ツール「Studio」
- **ベーステンプレート**：無料テンプレート「THE COFFEE APP」（1カラム構成）を使い、中身を入れ替える方法で進める。ユーザーはStudioの操作が初めてのため、空白から作ると挫折するリスクがある。

**ナビゲートの鉄則**：抽象的なCSS用語は使わない。「Studioの右パネルの〇〇の数値を〇〇にする」「このボックスをダブルクリックして文字を書き換える」など、初心者が迷わない具体的な画面操作で案内すること。

### 作業前の準備（最初に必ずやること）

Studioで画像を使うには、先にメディアライブラリへのアップロードが必要。作業開始前に以下の画像ファイルをStudioにアップロードしておくよう、ユーザーに案内してください。

| ファイル名 | 使いどころ |
|---|---|
| `hero-mother-baby.png` | ファーストビュー背景 |
| `about-nature.jpg` | 私たちについてセクション |
| `temori-handing.png` | てもりセクション |
| `temori-baby-hands.png` | てもりセクション |

### フォント

| 用途 | フォント名 |
|---|---|
| 見出し | Shippori Mincho（しっぽり明朝） |
| 本文 | Noto Sans JP または Kosugi Maru |

### アニメーション（全セクション共通）

スクロールに合わせて、各セクション・カードが下から上に「フワッ」と浮き上がるフェードイン。
- Y軸：10px
- 時間：0.8秒
- イージング：ease-out

### デザインルール

- 余白は広めにとる（文字を詰め込まない）
- 角丸はなし〜ごく小さく
- グラデーション・派手な影は使わない
- 1スライド・1セクションに伝えることは1つに絞る

### 【3】商品紹介：てもり

*てもりLPのS①〜S⑦を合体*

#### 3-1 ファーストビュー的な導入

- 画像：`temori-handing.png`
- キャッチ：`この子と、ずっと。`
- サブ：
```
生まれた日の7つの記録が、岩手の木に模様として刻まれる。
世界にひとつの、あなただけの木のかけら。
```

#### 3-2 コンセプト

- 見出し：`7つの記録が、木の模様になる。`
- 本文：
```
名前、誕生日、体重、身長、出生時刻、そのときの天気、生まれた場所。
この7つのデータを自然計算アルゴリズムに通すと、
その子だけの、世界にひとつの模様が生まれます。

木製品でも、写真でも、名前入りグッズでもない。
「生まれた瞬間」そのものが、かたちになるもの。
それがてもりです。
```

#### 3-3 成長に合わせた使い方（3ブロック）

```
赤ちゃんのうちは、感触あそびに。
舐めても安全な食品グレードのコートで仕上げているから、3ヶ月からそのまま手渡せます。

少し大きくなったら、絵合わせに。
葉の形を見比べて、同じ木を探す。自然と木の名前を覚えていきます。

大人になっても、そばに置いておける。
生まれた日の記録が刻まれた9枚は、いつまでも「あのとき」を手元に残してくれます。
```

#### 3-4 これを選ぶ理由（4カード）

```
01　世界にひとつ、その子だけのもの
同じ名前でも、同じ体重でも、生まれた瞬間が違えば模様は変わります。受注生産だからできる贈りもの。

02　「かわいい」で終わらない記念品
写真は劣化する。洋服はサイズアウトする。木は経年変化しながら、何十年も手元に残ります。

03　岩手の職人の手仕事
素材の調達から加工・研磨・梱包まで、すべて岩手の人の手が関わっています。

04　安心して渡せる品質
食品グレードの塗装、こどもPSCマーク申請予定。赤ちゃんに渡す前提で設計しています。
```

#### 3-5 模様が生まれる仕組み（4ステップ）

```
STEP 1　データを入力する
Webアプリに赤ちゃんの7つの情報を入力します。

STEP 2　自然計算で模様を生成する
アルゴリズムが数値を自然のパターンに変換。同じ模様は、二度と生まれません。

STEP 3　岩手の木に刻む
NC加工またはレーザー刻印で、9枚の葉形パネルそれぞれに刻みます。

STEP 4　手元に届く
9樹種の葉形パネルが揃ったセットとして、丁寧に梱包してお届けします。
```

#### 3-6 岩手の木である理由

- 見出し：`この子が生まれた土地の、木を使う。`
- 本文：
```
てもりの素材は、岩手県産材9樹種。
盛岡の森から切り出され、丁寧に加工された15mm厚の板を使っています。
9枚並べると、岩手の森が手の上に広がります。
```
- 樹種バッジ（横並びで配置）：`スギ　カラマツ　アカマツ　ナラ　ヤマザクラ　クリ　オニグルミ　イタヤカエデ　ケヤキ`
