## 現在開発中の商品（プロジェクト「とうめいしっぽ」木のおもちゃ）

### ① 商品概要（案）

| 項目 | 内容 |
|------|------|
| **商品名** | 検討中（候補：木の葉文／葉印／森のことばなど） |
| **形状** | 軸なし・丸みある葉形／樹種ごとにシルエットが異なる |
| **素材** | 岩手県産材フローリング材・15mm厚（9樹種） |
| **樹種** | スギ・カラマツ・アカマツ・ナラ・クリ（市産材5種）＋ヤマザクラ・オニグルミ・イタヤカエデ・ケヤキ（県産材4種） |
| **仕上げ** | シオン社・食器用木守りコート（舐めても安全） |
| **対象年齢** | 0〜2歳〜（出産祝い・誕生記念品として0歳から） |
| **加工方法** | NCルーター＋レーザー刻印 |
| **模様** | 赤ちゃんの7つのデータ（名前・誕生日・体重・身長・出生時刻・天気・出生地）を自然計算アルゴリズムで変換した唯一無二のパターン |
| **価格目標** | 9枚セット¥15,000（枚数・構成は検討中） |
| **最小遊べる枚数** | 6枚（握る・並べる・積む・転がす遊びが成立） |

### 商品の意義（3層構造）

1. **まず、おもちゃであること**：0〜2歳が実際に握り、触れ、遊べる
2. **次に、家族をつなぐもの**：赤ちゃんのデータが宿る唯一の模様。親が選び→子が育て→受け継がれる家宝へ
3. **そして、現在と未来を・森と人をつなぐもの**：岩手の森の木が家庭に届き、使うことが森を守る循環をつくる
