# 保護者向け配布資料 様式プロンプト

以下の仕様でA4縦のPDFを作成してください。

---

## 使用環境
- Python + reportlab
- 日本語フォント：`NotoSerifJP-VF.ttf`（本文）、`yumindb.ttf`（太字見出し）
  - どちらも `C:\Windows\Fonts\` にあります

---

## デザイン仕様

### カラー（2色）
| 役割 | カラーコード | 用途 |
|---|---|---|
| INK | `#2B2B2B` | 本文・見出し文字 |
| ACCENT | `#7A6352` | 罫線・装飾・署名（白黒印刷でグレーになる） |

### レイアウト
- 左右マージン：18mm
- 外枠：なし
- 背景色：なし（白）

### フォントサイズとジャンプ率
| 要素 | フォント | サイズ |
|---|---|---|
| メインタイトル | yumindb（太字） | 20pt |
| サブタイトル | NotoSerifJP | 11pt（ACCENT色） |
| セクション見出し | yumindb（太字） | 13pt |
| 本文 | NotoSerifJP | 9.5pt |
| ヒント項目 | NotoSerifJP | 10.5pt |
| 署名 | NotoSerifJP | 8.5pt（ACCENT色・右寄せ） |

### 行間
- 本文行高：7.2mm
- ヒント項目行高：8.0mm

### セクション間余白
- ページ全体のコンテンツ高さを事前計算し、残りの余白をセクション間に均等配分する
- タイトル直後のセクション間のみ、1行分（7.2mm）追加で空ける

---

## 構造テンプレート

```
【タイトル行】（中央揃え・太字・20pt）
【サブタイトル行】（中央揃え・ACCENT色・11pt）

　← タイトル後だけ1行多めに空ける →

【見出し①】（左アクセントバー＋太字13pt＋下線）
本文テキスト（1文1行で改行。折り返しあり）

【見出し②】
本文テキスト

【見出し③】
リード文
　｜ ヒント項目1（左縦線インデント）
　｜ ヒント項目2
　｜ ヒント項目3
締めの一文（本文サイズ）

───────────（ACCENT色の細線）

署名（右寄せ・ACCENT色・8.5pt）
```

### 見出しのデザイン
- 左端に幅3mm・高さ8mmのACCENTカラーの塗りつぶし矩形
- 見出しテキストは矩形の右5mmに配置
- 見出し下3mmに幅全体のACCENTカラー細線（0.5pt）

### ヒント項目のデザイン
- 左端から4mm内側に幅1.5mmのACCENTカラー縦線
- テキストは縦線の右4mmに配置
