# 工作教室 スプレッドシート整理 指示書

## 目的

現在の年度別・バラバラなExcelファイルを、以下の2種類に整理する。

1. **管理用マスターファイル**（1ファイル）：過去データの検索・集計用
2. **提出用ファイル**（年度別・1年度1ファイル）：依頼元に渡す年間計画書

---

## フォルダ構成

```
工作教室/
├── 管理用/
│   └── 工作教室マスター.xlsx         ← 1ファイルで全年度管理
└── 提出用/
    ├── R7_工作教室_提出用.xlsx        ← 令和7年度（依頼元別シート）
    ├── R8_工作教室_提出用.xlsx        ← 令和8年度（依頼元別シート）
    └── ...（年度ごとに追加）
```

---

## ① 管理用マスターファイルの作成

### ファイル名
`工作教室マスター.xlsx`

### シート構成（3シート）

---

### シート1：依頼元マスタ

**用途：** 依頼元の基本情報を一元管理する

| 列名 | 内容 | 例 |
|---|---|---|
| 依頼元ID | 連番（C001〜） | C001 |
| 依頼元名 | 正式名称 | 上米内児童センター |
| 略称 | ファイル名などに使う短縮名 | 上米内 |
| 種別 | 児童センター・地区センターなど | 児童センター |
| 担当者名 | 先方の担当者 | 〇〇様 |
| 連絡先 | 電話・メールなど | |
| 継続状況 | 継続中 / 終了 | 継続中 |
| 備考 | 特記事項 | |

**既存の依頼元（スクリーンショットより確認済）：**

| 依頼元ID | 依頼元名 | 略称 |
|---|---|---|
| C001 | 高松（正式名称に修正） | 高松 |
| C002 | 青山（正式名称に修正） | 青山 |
| C003 | 北松園（正式名称に修正） | 北松園 |
| C004 | 緑ヶ丘（正式名称に修正） | 緑ヶ丘 |
| C005 | 上米内児童センター | 上米内 |
| C006 | コピー（正式名称に修正） | コピー |

> **注意：** 正式名称は元ファイルを確認して入力してください。

---

### シート2：案件DB

**用途：** 全年度・全依頼元の実施案件を一覧で管理する（1行1案件）

| 列名 | 内容 | 例 |
|---|---|---|
| 年度 | 和暦（R5〜） | R7 |
| 依頼元ID | 依頼元マスタのID | C005 |
| 依頼元名 | 依頼元マスタから参照 | 上米内 |
| 回数 | 年度内の実施回数 | 1 |
| 実施日 | 日付（YYYY/MM/DD形式） | 2025/04/25 |
| タイトル | 作品名 | コラージュパズル |
| 内容 | 作業内容の説明 | コラージュの技法とラミネートを使ってオリジナルパズルを作る |
| 必要材料 | 当日使う材料一覧 | ラミネートフィルム、はさみ、のり |
| 参加人数 | 実績人数 | 12 |
| ステータス | 予定 / 実施済 / 見学済 / キャンセル | 実施済 |
| 備考 | 特記事項 | |

**移行対象データ：**
- `コピー工作教室まとめ`シートの全データ（R5〜R8）
- `R7上米内`シートの実施案件（回数・日程・タイトル・内容・材料）
- 他の依頼元シートがあれば同様に移行

**フィルタの設定：**
1行目をヘッダー行として固定し、Excelのテーブル機能（Ctrl+T）でテーブル化する。これにより年度・依頼元・ステータスでの絞り込みが可能になる。

---

### シート3：経費DB

**用途：** 全年度の依頼元別の預かり金・支出・残額を管理する（1行1依頼元1年度）

| 列名 | 内容 | 例 |
|---|---|---|
| 年度 | 和暦（R5〜） | R8 |
| 依頼元ID | 依頼元マスタのID | C001 |
| 依頼元名 | 依頼元マスタから参照 | 高松 |
| 預かり金 | 依頼元から受け取った金額 | 25000 |
| 支出合計 | 材料費などの合計 | 0 |
| 残額 | 預かり金−支出合計（自動計算） | =預かり金−支出合計 |
| 領収書提出 | 提出済 / 未提出 | 未提出 |
| 備考 | 特記事項 | |

**移行対象データ：**
- `R8_材料費`シートの各依頼元のデータ（R8年度分）
  - 高松：預かり金 25,000円
  - 青山：預かり金 16,000円
  - 北松園：預かり金 16,000円
  - 緑ヶ丘：預かり金 15,000円
  - 上米内：預かり金 5,000円
- 過去年度（R5〜R7）の材料費データも同様に移行

## ② 提出用ファイルの作成

### ファイル名の規則
`R{年度}_工作教室_提出用.xlsx`（例：`R8_工作教室_提出用.xlsx`）

### シート構成
依頼元ごとに1シートを作成する。シート名は略称を使用。

**例（R8の場合）：**
```
R8_工作教室_提出用.xlsx
├── 高松
├── 青山
├── 北松園
├── 緑ヶ丘
└── 上米内
```

### 各シートのフォーマット（R7上米内シートに準拠）

```
1行目：{依頼元正式名称}様
2行目：令和{X}年度　工作教室　実施計画
3行目：（空白）
4行目：毎回ご用意いただきたいもの　｜　講師：新田紗希
5行目：人数分：はさみ、のり、セロハンテープ　｜　おもちゃコンサルタント／公認心理師
6行目：あれば：カラーペン、色画用紙の端材
7〜11行目：諸注意の文章
12行目：（空白）
13行目：ヘッダー行（回数 | 実施日 | タイトル | 内容 | 必要な材料（1人12〜全制数迄））
14行目以降：実施案件データ
```

**列幅の目安（R7上米内シートに準拠）：**
| 列 | 内容 | 幅の目安 |
|---|---|---|
| A | 回数 | 狭め |
| B | 実施日 | 中程度 |
| C | タイトル | 中程度 |
| D | 内容 | 広め |
| E | 必要な材料 | 広め |

---

## ③ 移行の優先順位

作業量が多いため、以下の順で進めることを推奨：

1. **まず管理用マスターのシート構成（ヘッダー行）だけ作る**
2. **R8年度（今年度）の案件DBと経費DBを移行する**
3. **R8提出用ファイルを作成し、各依頼元シートを整える**
4. **余裕があればR7以前のデータを遡って移行する**

---

## ④ 注意事項

- 元のファイルは**削除せず必ずバックアップとして残す**こと
- 依頼元の正式名称は元ファイルを確認して正確に入力すること
- 日付はすべて`YYYY/MM/DD`形式（例：2025/04/25）で統一すること
- 経費DBの残額列は手入力せず、必ず計算式（`=C列-D列`など）を使うこと
- アイデアメモ（コピー工作アイデアシート）はObsidianへの移行を検討し、スプレッドシートには入れない

| 現在のシート名 | 移行先 |
|---|---|
