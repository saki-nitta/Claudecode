# フォルダ構造ルール

各フォルダ（01〜99）の中に `docs/` を作り、以下の構成で管理する：

```
docs/
  00_raw        ← 元データ（手を加えていないもの全部）
  01_converted  ← 読める形式に変換したもの（PDF→Word など）
  02_extracted  ← テキスト抽出・要約したもの
  03_final      ← 最終的に使う資料
```

対象フォルダ：
- 01_toumeishippo（事業・収益）
- 02_iwate_branch（岩手支部・非収益）
- 03_psychology（心理職）
- 04_common（共通・横断）
- 99_sandbox（実験）
