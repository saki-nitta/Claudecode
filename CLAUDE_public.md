### 柱④ おもちゃ開発（てもり / このは）の詳細

#### 商品概要
- 岩手県産材9樹種の葉形木のおもちゃ
- 赤ちゃんの誕生データ（名前・体重・身長・出生時刻・天気・出生地）をアルゴリズムで模様に変換
- 対象: 3ヶ月〜、9枚セット・価格¥13,000〜¥25,000（モード別）
- 素材: 15mm厚フローリング材、食器用木守りコート仕上げ
- 加工: NCルーター＋レーザー刻印

#### コンセプト
1. **フレキシブル**: 乳幼児の感覚遊びから絵合わせ・ゲームへと成長に合わせて変化
2. **タイムレス**: 性別・流行に左右されないデザイン。出産祝いに選びやすい
3. **サスティナブル**: 経年変化を楽しめる。世代を越えて使い続けられる

#### 価格体系（3モード）
| モード | 販売価格 | 製造コスト | 特徴 |
|---|---|---|---|
| レーザー刻印のみ | ¥13,000 | ¥735〜754/枚 | シンプル彫刻のみ |
| V溝NC加工 | ¥16,000 | ¥877〜896/枚 | 立体的な彫刻 |
| NC＋レーザー | ¥25,000 | ¥1,377〜1,396/枚 | フルカスタム |

## 運用ルール

### ファイル命名規則
```
YYYYMMDD_分類_内容_状態_版
```
- 分類: research / idea / concept / design / test / event / sales / doc
- 状態: raw / converted / extracted / draft / review / final

### セキュリティ
- `.env` や `~/.ssh` へのアクセス禁止
- プロジェクトフォルダ外のファイル変更禁止
- APIキー・パスワードはコードにハードコードしない

### docs フォルダ構成
- `00_raw`: 元データ（変更しない）
- `01_converted`: 読める形式に変換
- `02_extracted`: 要約・抜粋（主に参照）
- `03_final`: 完成物
