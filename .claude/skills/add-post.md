# /add-post — Instagram投稿ストックに追記する

## 対象ファイル

- 投稿ストック: `D:\Obsidian_Sync\リモート保管庫1\05_発信\コラム\Instagram\Instagram_投稿ストック.md`
- カウント管理: `D:\Obsidian_Sync\リモート保管庫1\05_発信\コラム\コラム管理.md`

## 手順

1. **投稿ストックを読む**（末尾のカテゴリ番号を確認するため）
2. **番号を決める**
   - ユーザーが「B-x」「C-x」など指定した場合はそれに従う
   - 指定がない場合はカテゴリの最後の番号+1にする（例：C-1があればC-2）
   - B系投稿はC系投稿の前に挿入する（カテゴリ順を維持）
3. **投稿を追加する**（`---` で区切り、以下のフォーマットで）

```
## XX-X「タイトル」

**本文：**
（本文）

**カテゴリ：** X（種別）
**タグ：** #タグ1 #タグ2
**ステータス：** 未投稿
```

4. **コラム管理.mdの未投稿カウントを更新する**
   - `媒体別ステータス` テーブルの Instagram 行を更新
   - 本数と投稿タイトル一覧を最新にする

## 使い方

ユーザーが投稿内容（タイトル・本文・カテゴリ・タグ）を貼ったら、確認なしで追加してカウントを更新する。
カテゴリ番号が既存と重複しそうな場合のみ確認する。
